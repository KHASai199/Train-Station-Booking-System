# Train-Station-Booking-System
# A desktop application built with C# and Windows Forms for managing train ticket bookings. It provides a complete user flow including login/sign-up, ticket booking, # train schedule viewing, and an admin panel for managing users and operations.
# Key Features:

# User authentication (Login & Sign Up)
# Ticket booking system
# Train schedule management
# Admin dashboard
# Live clock for train timing

# Tech Stack: C#, .NET, Windows Forms, Visual Studio
